Произведение матриц

Сумма матриц


Определение

Произведение матриц (С= АВ) — операция только для согласованных матриц А и В, у которых число столбцов матрицы А равно числу строк матрицы В: C = A × B


Свойства умножения матриц

  • (AB)C = A(BC)
  • A(B + C) = AB + AC
  • (A + B)C = AC + BC
  • λ(AB) = (λA)B
